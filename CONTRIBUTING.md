# Contributing to VAANI

Thank you for your interest in contributing to VAANI! We welcome all contributions.

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR-USERNAME/Vaani.git
cd Vaani
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Changes
- Follow Kotlin style guide
- Add comments for new code
- Test on Android device

### 4. Commit & Push
```bash
git add .
git commit -m "Add feature: your feature description"
git push origin feature/your-feature-name
```

### 5. Create Pull Request
- Go to GitHub
- Click "New Pull Request"
- Describe your changes
- Wait for review

## Code Style

- Use Kotlin idioms
- Follow Google's Kotlin style guide
- Max line length: 120 characters
- Use descriptive variable names

## Reporting Bugs

1. Check if bug already exists
2. Create issue with title: `[BUG] Your bug title`
3. Include:
   - Android version
   - Phone model
   - Steps to reproduce
   - Expected vs actual behavior

## Suggesting Features

1. Create issue with title: `[FEATURE] Your feature idea`
2. Describe the feature
3. Explain why it would be useful
4. Add mockup/example if possible

## Questions?

Open an issue with `[QUESTION]` prefix in the title.
